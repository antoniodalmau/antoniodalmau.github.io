Easily display interactive 3D models on the web & in AR.


<model-viewer> is a web component that makes rendering interactive 3D models - optionally in AR - easy to do, on as many browsers and devices as possible.

<!-- Import the component -->
<script type="module" src="https://unpkg.com/@google/model-viewer/dist/model-viewer.min.js"></script>
<script nomodule src="https://unpkg.com/@google/model-viewer/dist/model-viewer-legacy.js"></script>

<!-- Use it like any other HTML element -->
<model-viewer src="mug.gltf"></model-viewer>

glTF is a open standard file format for three-dimensional scenes and models. It supports 3D model geometry, appearance, scene graph hierarchy, and animation.
